Manage git hooks for automatic updates

Install or uninstall git hooks that automatically update the
dependency database after commits and merges.

```
git-pkgs hooks [flags]
```

### Options

```
  -h, --help        help for hooks
      --install     Install hooks
      --uninstall   Uninstall hooks
```
